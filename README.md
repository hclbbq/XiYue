## 框架介绍

(XiYue)基于SpringCloud的权限管理系统，采用最新的SpringBoot3.0 和 SpringCloud的轻量级微服务脚手架，权限系统套了一层系统模块表，非常适合一个系统管理名下好几个子系统权限的场景

采用SpringBoot+MybatisPlus+AntDesignVue+Vite 等更多优秀组件及前沿技术开发，注释丰富，代码简洁，开箱即用！
